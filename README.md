# gcloud-common-utils

Helpers for Google Cloud Storage and Pub/Sub.

## Install

From PyPI (if published):
```bash
pip install gcloud-common-utils
```

From source:
```bash
git clone https://github.com/your-org/gcloud-common-utils.git && cd gcloud-common-utils
poetry install
```

Authenticate (once):
```bash
gcloud auth application-default login
```
